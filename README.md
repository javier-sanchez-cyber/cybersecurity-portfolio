# Cybersecurity Portfolio — Blue Team / SOC

## Sobre mí

Perfil junior de ciberseguridad orientado a Blue Team/SOC, análisis de incidentes, phishing analysis, SIEM, documentación técnica y automatización básica con Python.

Actualmente estoy construyendo un portfolio práctico centrado en casos de uso reales o simulados de SOC: análisis de phishing, investigación de logs, reglas de detección, uso de MITRE ATT&CK, consultas KQL y automatización de tareas repetitivas.

## Objetivo profesional

Busco oportunidades junior, prácticas o roles de entrada relacionados con SOC, Blue Team, análisis de seguridad, respuesta a incidentes o soporte técnico con enfoque en ciberseguridad.

Mi objetivo es desarrollar experiencia práctica en entornos defensivos, mejorar mi capacidad de análisis y aportar valor en tareas operativas de seguridad.

## Metodología de trabajo

Cada proyecto sigue una estructura orientada a análisis profesional:

1. Contexto del escenario.
2. Objetivo del análisis.
3. Fuente del escenario.
4. Metodología.
5. Evidencias observadas.
6. Análisis técnico.
7. Evaluación de riesgo.
8. Recomendaciones.
9. Limitaciones.
10. Lecciones aprendidas.

## Nota ética

Todos los proyectos de este repositorio tienen fines educativos y defensivos.

No se incluyen datos personales reales, credenciales, enlaces maliciosos activos, muestras ejecutables de malware ni información sensible de terceros.

Cuando se utilicen indicadores, URLs o evidencias, serán simulados, sanitizados o neutralizados.
